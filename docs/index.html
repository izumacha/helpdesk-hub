HelpDesk Hub ドキュメント

このプロジェクトの設計・運用ドキュメントの目次です。最初に読む場合は overview から。

入口（最初に読む）

  • overview— PM・非エンジニア向けのプロジェクト全体解説

設計

  • requirements— 機能要件・スコープ・ステータス遷移の正本
  • architecture— システム構成、レイヤ分担、データフロー、SSE 通知
  • er-diagram— ER 図とテーブル定義
  • screen-flow— 画面遷移図

運用・品質

  • security— セキュリティ・堅牢性メモ
  • version-integration— 差分調停ルール（要件 vs 実装のズレを解決する基準）
  • implementation-notes— 実装上の補足メモ

進捗・レビュー

  • issue-backlog— 優先度ベースの実装バックログ
  • github-issues— GitHub Issue 登録用の Issue ドラフト集
  • pr-review-report— PR レビューレポート
  • self-review— セルフレビュー記録
